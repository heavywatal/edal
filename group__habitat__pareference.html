double Individual::resource_overlap ( const Individual &  other) const

Honest \(C(I, J)\) for competition and mating (very slow, unused)

Parameters
otherindividual to interact
Returns
\(C(I, J)\)
Return values
1for individuals with identical preferences

Competition and mating frequencies decrease with increasing the total difference of the resource consumption, \(F(u,v) \Xi(x_0,x_1|u,v) W(y_0,y_1|u,v)\), over the resource range.

\[ C(I,J) = \exp[-c_0\{\iint F(u,v)(\Xi(I|u,v) W(I|u,v) - \Xi(J|u,v) W(J|u,v)) du dv\}^2] \]
